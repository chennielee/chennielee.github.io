02 — Environment (State / Action / Reward)

Scope (toy version)

  • 1 satellite
  • N ground stations (start with N=3)
  • Discrete time steps

State (observation)

  • backlog: amount of data waiting (normalize 0..1)
  • battery: remaining energy budget (0..1)
  • gs_available[i]: is GS i visible now? (0/1)
  • link_quality[i]: quality of link to GS i (0..1)

Action

Choose one ground station: a ∈ {0..N-1}

Transition (what happens)

  • If chosen GS is available: send some data, backlog decreases, battery decreases.
  • If unavailable: sending fails (penalty), backlog stays.
  • Optionally: link quality changes over time (simple random or sinusoid).

Reward (starter version)

reward = sent_data − λ * energy_cost − fail_penalty

  • sent_data: e.g., proportional to link_quality if successful
  • energy_cost: constant or proportional to sent_data
  • fail_penalty: fixed negative value if action fails
  • λ (lambda): trade-off knob (start with 0.1)
